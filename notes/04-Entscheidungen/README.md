# 04 — Entscheidungen (leicht)

Schnelle Entscheidungen, die du festhalten willst, aber die (noch) kein
formales ADR brauchen. Vorlage: [[templates/Entscheidung]].

**Wann formal nach `docs/adr/`?**
- Betrifft Architektur, Sicherheit, Recht oder Revenue.
- Hat langfristige Konsequenzen / ist schwer rückgängig zu machen.
- Andere müssen es nachvollziehen können.

Sonst reicht eine Notiz hier.
